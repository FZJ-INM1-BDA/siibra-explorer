Selecting template

On the first page of interactive atlas, you can see the list of templates. Click to select one of them.

Also, you can select or change template from top menu. Open the template selection list and select desired template.

Selecting Parcellation

You can select parcellation from top menu. Open the parcellation selection list and select desired parcellation.

Selecting regions

Open the parcellation selection list and select the desired parcellation. You can select regions with two ways

Select region from top menu. It is possible to search for a region and click on it. (See picture 1)

By double-clicking on it in orthogonal 2D cross sections or 3D volume of the brain volume. (See picture 2)

Search the knowledge graph (KG) for datasets.

The interactive viewer is natively setup to query KG for any datasets related to specific parcellation area(s).

Single area search

To conduct a single area search, right click on any parcellation area. In the context menu, select Search KG for ... and a data browser window will be launched, fetching data sets associated with this area.

Multi area search

To conduct a multi area search, double click as many areas in the search scope as necessary. When the region selection is complete, right click the viewer and select Search KG for ... selected regions, similar to single area search, a data browser will be launched, fetching data sets associated with all of the areas selected.

Navigation

Change coordinates

There are several ways to change navigation position (coordinates of intersection of planar views):

1. Drag orthogonal 2D cross sections with a mouse to desired position.

2. Type coordinates in navigation field (input field located at left bottom side).

3. Scrolling with mouse wheel while holding a ctrl key will scroll through the slices in the corresponding 2D view under the mouse cursor. Simultaneously holding shift key in addition to ctrl key will increase the speed of scrolling to 10 slices at a time instead of one.

Change zoom level

Mouse scroll wheel is used to increase or decrease zoom level of 2D cross sections and 3D overview.

Rotation (oblique slicing)

Arbitrary orientation of planar views (oblique slicing) can be achieved by dragging orthogonal 2D cross sections while pressing the shift key.
